Acessibilidade do Site

Está página lista e descreve o funcionamento de todos os recursos de acessibilidade que o site possui.

Atalhos do site

Abaixo estão listados os atalhos do site que podem ser usados pelo teclado:

Usando o Google Chrome, Microsoft Edge, Opera ou Safari:

  • Alt + 1: Direciona o foco diretamente para o conteúdo da página.
  • Alt + 2: Direciona o foco para o menu principal do site.
  • Alt + 3: O direciona para a página mapa do site.
  • Alt + 4: O direciona para a página Acessibilidade do site, que é a página atual.
  • Alt + 5: Direciona o foco para o menu de navegação auxiliar.

Usando o Safari no Mac OS:

Ao invés de pressionar Alt+Número você deve pressionar Ctrl+Alt+Número.

Usando o Mozilla Firefox:

Caso esteja usando o mozilla firefox no Windows você deve, ao invés de pressionar Alt+Número você deverá pressionar Shift+Alt+Número. Caso esteja usando no Mac OS você deverá pressionar Ctrl+Alt+Número.

Utilizando versões antigas do Opera:

Se caso o seu navegador for o Opera e sua versão for a 12.1 ou anterior, ao invés de pressionar Alt+Número você deverá pressionar Shift+Esc+Número.

Atalhos do Questionário:

Enquanto estiver na página do questionário os atalhos ainda serão as mesmas combinações, mas com funções diferentes:

  • Alt + 1: Direciona o foco para a questão atual.
  • Alt + 2: Envia o foco para o botão de pular a questão.
  • Alt + 3: Envia o foco para o botão de confirmar a questão.
  • Alt + 4: Envia o foco ao menu com as opções de acessibilidade.
  • Alt + 5: O direciona para a página inicial.

Botões contraste

Ao lado direito do menu principal há dois botões que alteram o contraste de cores do site.

  • Altera o contraste do site para cores mais escuras
  • Volta o contraste do site para as cores padrões

Botões alterar a fonte

Também ao lado direito do menu principal há três botões, A+, A° e A-, que alteram o tamanho da fonte do conteúdo do site:

  • : Aumentar o tamanho das fontes do site.
  • : Volta às fontes ao tamanho padrão.
  • : Reduz o tamanho das fontes do site.

Zoom

O navegador dá a possibilidade de ampliar o site, pelos comandos:

  • Aumentar o zoom: Basta pressionar Ctrl e a tecla + (mais) juntas.
  • Voltar ao zoom padrão: Basta pressionar Ctrl e a tecla 0 (zero) ao mesmo tempo.
  • Reduzir o zoom: Basta pressionar Ctrl e a tecla - (menos).

Pode-se usar o zoom juntamente com os botões de aumentar a fonte para facilitar a visualização do conteúdo.

Menu navegação auxiliar

Ele é um menu oculto que aparece no topo da tela ao receber o foco. Nele há três links:

  • Ir para o menu: Envia o foco ao menu principal.
  • Ir para o conteúdo da página: Envia o foco direto ao conteúdo da página.
  • Ir para o mapa do site: O direciona para o mapa do site.

Mapa do site

Essa região reúne, de maneira organizada, vários links que o envia para qualquer região específica do site. Para acessá-lo você pode: utilizar o atalho Alt+3(ou outro comando de acordo com o navegador), acessar o link no menu principal, no menu de navegação auxiliar ou acessar o link a seguir: Mapa do site.

Código fonte

O código fonte desse site foi cuidadosamente feito para que fosse funcional, acessível, responsivo e que todo o site possa ser acessado pelo teclado.

Leitores de tela

O site está adaptado para uso de leitores de tela, que são ferramentas usadas por deficientes visuais para fazer uma leitura do que há na tela e descreve-los para o usúario em forma de audio. Abaixo há dois links para download dos seus respectivos leitores de tela.

  • NVDA: Leitor de tela totalmente gratuito e utilizado para os testes de desenvolvimento do site.
  • Jaws: Um leitor de tela altamente eficiente, mas para utiliza-lo por longos períodos de tempo deve-se adiquirir uma licença paga. Deste modo, torna-se inacessível a muitos. Há uma versão de teste que pode ser utilizada por um certo período de tempo.
